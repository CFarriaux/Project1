    Chocobooks API

    How to get access to our review data

    Accessing the API

    Access book information at the following url:

    http://127.0.0.1:5000/api/books/isbn

    Note: You must be logged in to access the API

    API data format

    You will receive the data in the following format:

    { "title": "Memory", "author": "Doug Lloyd", "year": 2015, "isbn": "1632168146", "review_count": 28, "average_score": 5.0 }

    Note: When no reviews are available, "average_score" will return "N/A"
